# College Guidance Website

A web app to guide college students on their journey.

## Setup
1. Install Node.js (if not installed).
2. Run `npm install` in the root folder.
3. Run `npm start` to launch at http://localhost:3000.

## Features
- Personalized suggestions based on year, experience.
- Info on branches like BTech, BBA.
- Research and coding resources.
- Simple live chatbot.
